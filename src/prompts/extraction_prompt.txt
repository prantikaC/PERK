- PaperStatus: statusType, statusDate
- Meeting: meetDate, meetTime, meetAgenda, meetLink

- uses (Method, Dataset)
- attends (Person, Meeting/Conference)

1. When you see a person mentioned in the body who is already in the header list:

3. For relations involving persons from headers, use their existing IDs

6. Each entity must have: id, type, and properties
7. Each relation must have: start_id, end_id, relation and context (the quote where this relation is found)
8. Output ONLY the JSON object, NO other text, NO markdown code fences
